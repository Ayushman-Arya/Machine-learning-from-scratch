implementing core concepts of machine learning by itself using python and numpy
# Linear Regression from Scratch

## Objective

Implement Linear Regression without using scikit-learn.

## Algorithms Implemented

- Batch Gradient Descent
- Stochastic Gradient Descent

## Dataset

House price prediction using area as a feature.

## Concepts Learned

- Cost Function
- Gradient Descent
- Learning Rate
- Feature Scaling
